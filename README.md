# Sts2 Spire Forge — Asset Mirror

Public image mirror for the **Sts2StartingDeckSelect** + **Sts2ClassAbilities**
mods. Used by the [Spire Forge wiki](https://wikiwiki.jp/sts2_starter/) for
inline image references via `raw.githubusercontent.com` URLs.

The mod source code itself is **not** open source — only the in-game art
assets (mod-original, generated for these projects) are mirrored here.

## Layout

```
cards/      Card portraits (mod-exclusive Basic cards)
relics/     Starter relic icons (large)
abilities/  Ult emblem icons (large)
tacticals/  Tactical emblem icons
powers/     Power icons
```
